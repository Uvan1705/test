## 📖 Описание программы
GUI-приложение для хранения информации о фильмах. Позволяет добавлять фильмы с названиями, жанрами, годом выпуска и рейтингом. Реализована фильтрация фильмов по жанру и году, а также сохранение всей коллекции в JSON-файл.

### ⚙️ Возможности
✅ Ввод полей: название, жанр, год выпуска, рейтинг
✅ Добавление новых фильмов в коллекцию
✅ Просмотр и фильтрация фильмов
✅ Фильтр по жанру
✅ Фильтр по году
✅ Сохранение и загрузка коллекции фильмов в формате JSON
✅ Проверка корректности вводимых данных (год — число, рейтинг — от 0 до 10)

## 👀 Примеры использования
-Введите название: Интерстеллар
-Жанр: Фантастика
-Год: 2014
-Рейтинг: 8.6
-Нажмите «Добавить фильм».
-Результат: Фильм появится в таблице.

## Фильтрация
-В поле “Фильтр по жанру” введите Фантастика
-Нажмите “Применить фильтр”
-Вывод: В таблице будут только фильмы жанра “Фантастика”.
### 💾 Сохранение и загрузка
-Данные фильмов автоматически сохраняются в файл movies.json.
-При перезапуске программы коллекция фильмов автоматически загружается из файла.
